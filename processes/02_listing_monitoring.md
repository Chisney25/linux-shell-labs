## Listing Processes with ps

The `ps` command provides a snapshot of running processes.

```bash
ps aux
```
Key columns

- USER – owner of the process
- PID – process ID
- %CPU – CPU utilization
- %MEM – memory utilization
- COMMAND – executed command

---

DevOps usage:
- Identify runaway processes
- Confirm whether services are running
- Validate correct process ownership

---

## Real-Time Monitoring with top

```bash
top
```
The top command provides a live view of system activity, including:

- CPU usage
- Memory consumption
- Running processes
- Load averages

Press q to exit.
