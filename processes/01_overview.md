# Linux Processes and Signals

Linux process management is a core operational skill in DevOps. It is used to diagnose performance degradation, terminate faulty workloads, and maintain system stability in production environments.

---

## Process Fundamentals

A process is a running instance of a program. Every process has:
- A unique Process ID (PID)
- An owning user
- CPU and memory consumption
- A lifecycle state

Understanding processes enables effective troubleshooting and incident response.
