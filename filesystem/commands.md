# Linux Filesystem & Navigation

## pwd
Displays the current working directory. Prevents path-related errors in automation scripts.

## ls / ls -l
Lists directory contents. The `-l` flag reveals permissions and ownership, which directly impact application behavior.

## mkdir
Creates directories. Used during provisioning and deployment automation.

## touch
Creates an empty file or updates timestamps. Commonly used for configuration scaffolding and logging.

## cp / mv / rm
Core file operations required for CI pipelines, backups, and cleanup jobs.
